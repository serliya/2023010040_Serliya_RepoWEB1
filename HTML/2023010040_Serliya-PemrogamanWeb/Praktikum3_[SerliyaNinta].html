Ayo kita belajar materi HTML bersama

HTML (Hyper Text Markup Language) adalah sebuah bahasa pemrograman yang digunakan untuk membuat halaman website. HTML menjadi dasar dari sebuah halaman website. sebuah file HTML di di simpan dengan ekstensi .html dan dapat di eksekusi atau diakses menggunakan web browser (Google Chrome, Mozilla Firefox, Opera, dan lain-lain). Untuk membuat sebuah website tidak cukup hanya menggunakan HTML, diperlukan bantuan CSS, JavaScript dan PHP untuk membuat website yang dinamis. Halaman website yang dibuat hanya menggunakan HTML maka halaman website tersebut di sebut halaman statis karena tidak memiliki aksi atau fungsifungsi yang dapat mengelola website.
